Track files

For each sequencing library, for each annotation family, ncPRO-seq generates compressed track files in both http://genome.ucsc.edu/goldenPath/help/bedgraph.htmlbedgraph and http://genome.ucsc.edu/goldenPath/help/bed.htmlbed formats to describe read mapping in sense and antisense direction of annotation items respectively.

As we have listed in 3.1, there is an option called GENOME_TRACK_OPTIONS which allow users to select different subsets of reads mapped in the genome to generate BedGraph formatted track files.

In BedGraph formatted track files, read coverage is computed for each genomic position.

All track files outputed from ncPRO-seq can be directly uploaded to http://genome.ucsc.edu/index.html UCSC browser / other genome browser for visualization.
