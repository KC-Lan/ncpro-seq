User Custom gff files

For each annotation family including different repeat/Rfam families and other custom annotations, ncPRO-seq creates a table file showing read coverages of each single family member in all sequencing libraries. If some annotations are not available, or if the user want to search for reads annotated in other genomcis regions, the profiling can also be done on custom gff files. As for the input files, the full path of the gff files has to be set. Several custom gff files can be specified.
