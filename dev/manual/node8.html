Required softwares

The ncPRO-seq pipeline requires the following additional softwares. Please refer to section 7 for more details :

  • The http://bowtie-bio.sourceforge.net/manual.shtml Bowtie Aligner (v2.0) [#!Langmead2009!#] to align the reads from smallRNA-seq data.

  • The http://www.r-project.org/ R (v2.12.0) [#!Rcitation!#].

  • The R and http://www.bioconductor.org/ BioConductor [#!Robert2004!#] packages: http://www.bioconductor.org/packages/2.2/bioc/html/seqLogo.html seqLogo [#!OliverseqLogo!#], http://www.bioconductor.org/packages/2.6/bioc/html/girafe.html girafe [#!Joern2010!#], http://cran.r-project.org/web/packages/RColorBrewer/index.html RColorBrewer [#!Erich2011!#], http://ggplot2.org/ ggplot2 [#!Wickham2009!#], http://cran.r-project.org/web/packages/reshape2/index.html reshape2 [#!Wickham2007!#] and http://cran.r-project.org/web/packages/gridExtra/ gridExtra.

  • The http://code.google.com/p/bedtools/ BEDTools suite (v2.15.0) [#!Quinlan2010!#], and the http://xfer.curie.fr/get/nil/2Rpz9IfJKcZ/BEDTools_MapCount_ColorTag.tar.gz bamMapCount program to address common genomic tasks such as finding feature overlaps and grouping same features.

  • The http://samtools.sourceforge.net/ SAMTools suite [#!Li2009!#] to handle SAM and BAM format.

  • The http://www.imagemagick.org/script/index.php Convert/ImageMagick utilies to format files and images.

  • In order to use the end-user interface, a local server with php such as apache is needed.

  • For Mac OS X user, make sure that you have http://developer.apple.com/xcode/ Xcode and command line tools for Xcode installed in your system.
