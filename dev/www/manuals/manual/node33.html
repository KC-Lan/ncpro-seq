miRNAs read proportion

In this step, abundant reads mapped in mature miRNA regions are counted, and plotted as the proportion of all mapped reads in the genome. The annotation file of mature miRNA is generated using files from miRBase [8] detailed in 6.3. Each miRNA count is calculated using the intersection of the reads alignement with the mature positions (see intersectBed program from BEDTools).
