Run the analysis

Finally, after setting the different parameters, the ncPRO-seq pipeline can be executed. A random url will be generated (Figure 8) to visualize the output report (see section 4). If specified, an email will be send to the user at the end of the analysis.
Please, note that in the current version the report will be updated only at the end of the analysis.
