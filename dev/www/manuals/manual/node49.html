Enabling CGI using XAMPP

If not well configured, the server can send a "Error 403 Access forbiden" during the CGI execution.
This could happen for instance with the file browser system of ncPRO-seq. To correct the error, be sure that the httpd.conf file of XAMPP is well configured. For instance, on MacOS, edit the file /Application/XAMPP/etc/httpd.conf and add the following parameters. 
<Directory "/Applications/XAMPP/xamppfiles/cgi-bin/">
    AllowOverride None
    Options Indexes FollowSymLinks MultiViews ExecCGI
    Order allow,deny
    Allow from all
</Directory>
This will enabling the CGI to be executed.
