tRNA annotation

The tRNA annotation file is generated from tRNA gtf files from UCSC [4] using custom scripts, detailed in readme file in annotation/prepareAnnotation folder. There is a feature in "attributes" column called "Type_Name" which indicate the type of tRNA. Example of "attributes" column:

ID=chr1.tRNA1547;Type_Name=tRNA-GluTTC
